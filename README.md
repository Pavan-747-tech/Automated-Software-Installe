# Automated Software Installer (Windows)

Simple CMD-based automation script for installing commonly used IT support software on Windows systems using Winget.

## Software Installed
- Microsoft Office (Microsoft 365)
- Google Chrome
- Adobe Acrobat Reader
- 7-Zip
- Microsoft Teams
- UltraViewer
